# 学而思云盘
可以让你上传/下载
利用学而思的上传API将文件存储到学而思服务器之中


![屏幕截图 2022-11-13 192405](https://livefile.xesimg.com/programme/python_assets/dcff06956a375384afcc4ab0d7a90437.png)

# 上传
可以将文件上传至云盘
PS：你需要复制上传后的链接


# 下载
可以下载存储在该云盘中的文件

# 储存空间
容量无限，全部储存在学而思的服务器中

# 分享
上传之后的链接就是你的分析链接。他可以分享给任何人，不会限制访问
